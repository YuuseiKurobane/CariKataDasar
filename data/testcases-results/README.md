# Generated regression cases

`_combined_testcases.txt` is generated automatically by `npm test` from every
text file in `data/testcases-dump/`.

Do not edit the generated file directly. Add or update a source dump and
run `npm test` instead. It performs materialization before running the test
suite so every accepted dump row remains protected. Strict-prefix plus
fuzzy-tail notation is preserved from the source dumps.
